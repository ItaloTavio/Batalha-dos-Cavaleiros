Batalha dos Cavaleiros

 Este projeto é o Trabalho Final da disciplina de Introdução a Computação, desenvolvido durante o 1º Semestre do curso de Sistemas de Informação
da Esamc. O objetivo foi criar um sistema de combate por turnos automatizados, com visualização no terminal, em linguagem C, aplicando técnicas de estruturação de código e modularização.

🛠 Tecnologias e Conceitos

-Linguagem: C

-Conceitos Aplicados:

- Lógica condicional e fluxos de decisão.
- Modularização de funções.
- Manipulação de strings.
- Estruturação de algoritmos.
